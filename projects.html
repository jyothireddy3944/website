Projects

American Airlines – Azure Data Lake Integration

Designed and implemented an enterprise-grade ETL framework using Talend and Matillion on Azure, integrating data into Snowflake with full support for batch and real-time ingestion.

  • Built modular dbt models to power ML pipelines and BI layers.
  • Integrated Azure Functions + Event Hubs for real-time streaming into Snowflake.
  • Optimized Databricks Spark jobs with partitioning and tuning.
  • Enabled CI/CD workflows via Azure DevOps and Git.
  • Applied governance with Alation and Collibra.

PepsiCo – GCP Data Platform Modernization

Led the migration of Oracle and SAS ETL processes to Google Cloud Platform, building scalable, automated pipelines using Dataproc, BigQuery, and Pub/Sub for real-time insights.

  • Converted legacy workflows to PySpark-based Dataproc jobs.
  • Triggered DAGs in Composer via Pub/Sub for pipeline automation.
  • Used Terraform for GCP infrastructure provisioning and IAM role setup.
  • Improved Tableau performance with BigQuery partitioned datasets.
