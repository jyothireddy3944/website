Detailed Projects

American Airlines – Enterprise Azure Data Lake Integration

You led the development of a high-throughput data engineering platform on Azure, building ETL pipelines using Talend and Matillion to extract structured and semi-structured data into a Snowflake warehouse. The architecture followed medallion principles (bronze/silver/gold) to improve semantic clarity and maintain governance layers.

  • Automated modular dbt models to support ML and BI reporting layers.
  • Implemented real-time ingestion using Azure Event Hubs and Functions into Snowflake.
  • Orchestrated dynamic Matillion pipelines with Python parameterization.
  • Boosted Spark job performance in Databricks with partitioning and caching.
  • CI/CD enabled via Azure DevOps, Git, and Maven.
  • Integrated data governance tools: Collibra, Alation.
  • Enabled Power BI dashboards with row-level security.

Tech Stack: Azure Data Factory, Databricks, Snowflake, Matillion, Talend, dbt, Spark, Python, Azure Functions, Event Hubs, Power BI

PepsiCo – GCP Dataflow & Migration Modernization

At PepsiCo, you redesigned legacy ETL workflows and migrated them from Oracle and SAS systems into a scalable, modern stack on Google Cloud Platform. The project emphasized real-time data processing and cost-effective infrastructure.

  • Built PySpark ELT workflows on Dataproc and BigQuery with optimized partitioning.
  • Migrated legacy Oracle/SAS ETL to Composer-triggered Airflow jobs.
  • Deployed secure CI/CD infrastructure using Terraform and GCP IAM.
  • Linked Pub/Sub to trigger real-time workflows with Cloud Functions.
  • Improved Tableau performance by tuning BigQuery views and datasets.

Tech Stack: GCP (BigQuery, Dataproc, Pub/Sub, Composer), Python, Airflow, Terraform, Dataprep, Tableau, Oracle SQL

AdventHealth – AWS-Powered Clinical Data Exchange

You built and managed HIPAA-compliant, real-time clinical data pipelines on AWS. These pipelines ensured accurate ingestion, transformation, and analysis of electronic health records using FHIR and HL7 standards.

  • Designed real-time ETL with AWS Kinesis, Lambda, and SQS.
  • Developed FHIR-compliant APIs for care coordination and interoperability.
  • Migrated on-premise data to AWS S3 and Redshift for scalability.
  • Standardized metadata and lineage using Talend/Informatica.
  • Built dbt models in Snowflake for clean clinical data layers.
  • Delivered Power BI dashboards with patient flow and quality metrics.

Tech Stack: AWS (Lambda, Kinesis, Redshift, SQS), Snowflake, Talend, Informatica, SSIS, FHIR, HL7, Python, Power BI
